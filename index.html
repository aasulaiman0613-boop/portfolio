Server-Authoritative NPC AI (Roblox)

Server-side perception, targeting, navigation, and damage validation.

AI
  • Architecture: Fully server-authoritative AI. The server owns movement, perception, target selection, pathing, and damage.
  • Modular systems: Vision, hearing, navigation, combat, and stimulus processing run inside a controlled update loop with throttled decisions to avoid jitter.
  • Vision: True line-of-sight with field-of-view checks and raycasts, with immediate loss and fast reacquire when visibility changes.
  • Hearing: Directional, probabilistic noise events (distance and type weighted) with intentional position offsets, so the NPC investigates an area instead of locking onto exact coordinates.
  • Movement and combat: PathfindingService with tuned agent parameters, repath intervals, stuck recovery, intentional stop distance, plus server-run close-range attacks with cooldowns and a touch-kill safeguard.
  • Anti-exploit: Server validates all targeting and damage, collision groups prevent player blocking, and unstable humanoid states are disabled for consistent physics.
