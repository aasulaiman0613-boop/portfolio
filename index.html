KidLang: Custom Programming Language and Transpiler

Language / Compiler

A fully functional programming language with a Python front-end that transpiles to high-performance C++17. Built to bridge beginner-friendly syntax and machine-level execution.

  • Tech: Python (lexer/parser), C++17 runtime (std::variant), Windows shell build automation.
  • Key innovation: Custom type system (kl::Val) enabling dynamic types in a statically typed backend.
  • Outcome: Handles complex loops and nested logic; validated for architectural soundness.
